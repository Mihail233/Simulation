# Проект «Симуляция»


Пошаговая симуляция двумерного мира в стилистике замка. В мире есть охотники за монетами и призраки. Охотники собирают монеты — основной ресурс. Призраки преследуют охотников. Также на карте размещены статичные объекты (стены и другие препятствия), которые занимают клетки и ограничивают передвижение. Каждый шаг симуляции обновляет перемещение существ и состояние мира по заданным правилам.
Приложение написано на языке Java в ООП-стиле. В поиске пути используется алгоритм поиск в ширину.

## Основные правила
- Игра имеет консольный интерфейс.
- Перед стартом программы пользователь должен ввести размеры карты,  и сложность замка (чем выше сложность, тем больше количество препятствий) после карта автоматически заполняется существами.
- Призраки преследуют и атакуют собирателей монет, а собиратели перемещаются по карте, собирая монеты.
- Существа имеют скорость (сколько клеток может пройти за 1 ход) и силу атаки.
- После каждой атаки показатель здоровья у атакуемого - уменьшается. У призраков показатель здоровья не меняется.
- Все существа двигаются по очереди, карта обновляется каждый круг.
- После каждого круга (хода всех существ) при низком количестве монет на карте появляются новые монеты.
- Симуляция заканчивается, если на карте не осталось собирателей монет или призраков.

### Для успешного запуска приложения:
- откройте терминал
- перейдите в папку Simulation
- скомпилируйте программу
    - `javac -d out $(find src -name "*.java")`
    - `jar -cvfm simulation.jar resource/META-INF/MANIFEST.MF -C out .`
- запустите программу `java -jar simulation.jar`
